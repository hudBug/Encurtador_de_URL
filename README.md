# Encurtador de URLs feito em Python
Para roda-lo são necessárias as bibliotecas pyshorteners, customtkinter e pyperclip.
Foi criada uma Interface Visual (GUI) para organizar e permitir a manipulação das URLs de forma mais confortável que um Console, foi inserido um bloco try...except para evitar o fechamento repentino da janela caso o usuário digite algo errado ou não seja possível encurtar a URL.
